# esman Javascript ES6

<h3>Установка</h3>

<h5>- Сервер node js</h5>

<div>
  git clone git@github.com:webivan1/esman.git ./ <br />
  cd src <br />
  npm istall <br />
  npm start
</div>

<h5>Yii2 install</h5>

<div>
  git clone git@github.com:webivan1/esman.git ./ <br />
  cd backend <br />
  php composer.phar self-update <br />
  php composer.phar update <br />
  cd ../src <br />
  npm webpack --watch
</div>
